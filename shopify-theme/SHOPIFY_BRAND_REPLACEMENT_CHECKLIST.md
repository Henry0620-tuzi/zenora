# Shopify 品牌替换清单

这份清单用来把当前演示品牌 `ZENORA` 替换成你的真实品牌。

## 一、先替换品牌基础识别

优先替换这些位置：

1. 店铺名
2. Logo
3. 品牌主色
4. 品牌辅助色
5. 品牌语气
6. 客服邮箱

对应位置：

- Shopify 后台 `Store details`
- 主题设置 `settings_schema.json`
- Header / Footer section
- 首页 Hero 文案

## 二、替换品牌文案

重点替换：

1. 首页 hero 标题
2. 首页副标题
3. 首页 trust strip
4. About 页品牌故事
5. FAQ 页物流和退货承诺
6. 商品页 fit note / shipping note / market badge

建议先统一这 4 个口径：

1. 品牌风格关键词
2. 目标人群
3. 目标市场
4. 价格带

## 三、替换商品与集合结构

建议先替换：

1. Collection 名称
2. Product title
3. Product description
4. Product media
5. Product tags
6. Product metafields

## 四、替换市场与运营信息

至少检查：

1. 支持国家
2. 币种
3. 语言
4. 运费
5. 退货政策
6. 税费/关税提示

## 五、联调前最终确认

确认这些已经不是演示内容：

1. `ZENORA`
2. `support@zenora.com`
3. 演示型价格带文字
4. 演示型商品名
5. 演示型 About 文案
